# KeenKeeper - Friendship Management System

**KeenKeeper** একটি আধুনিক ওয়েব অ্যাপ্লিকেশন যা আপনাকে আপনার বন্ধুদের সাথে যোগাযোগের হিসাব রাখতে এবং বন্ধুত্বের লক্ষ্য পূরণ করতে সাহায্য করে। এটি Figma ডিজাইন অনুসরণ করে তৈরি করা হয়েছে এবং এতে বন্ধুদের তালিকা দেখা, বিস্তারিত তথ্য জানা এবং ইন্টারঅ্যাকশন টাইমলাইন ট্র্যাক করার সুবিধা রয়েছে।

## লাইভ ডেমো
keenkeeper-alomgir.netlify.app

##(Key Features)

*   **ডায়নামিক ফ্রেন্ড লিস্ট:** JSON ডাটা থেকে বন্ধুদের প্রোফাইল কার্ড আকারে প্রদর্শিত হয়, যেখানে তাদের বর্তমান স্ট্যাটাস (Overdue, On-track, Almost due) অনুযায়ী কালার পরিবর্তন হয়।
*   **কুইক চেক-ইন এবং টাইমলাইন:** ফ্রেন্ড ডিটেইলস পেজ থেকে সরাসরি কল, টেক্সট বা ভিডিও কলের এন্ট্রি দেওয়া যায়, যা তৎক্ষণাৎ একটি 'টাইমলাইন' তৈরি করে। প্রতিটি অ্যাকশনে সুন্দর টোস্ট নোটিফিকেশন দেখা যায়।
*   **ফ্রেন্ডশিপ অ্যানালিটিক্স (Stats):** Recharts ব্যবহার করে একটি পাই চার্টের মাধ্যমে আপনার সব ইন্টারঅ্যাকশনের (Call, Text, Video) একটি সুন্দর ভিজ্যুয়াল রিপোর্ট দেখা যায়।
*   **ফুল রেসপন্সিভ ডিজাইন:** অ্যাপ্লিকেশনটি মোবাইল, ট্যাবলেট এবং ডেস্কটপ—সব ডিভাইসেই নিখুঁতভাবে কাজ করে।

##(Technologies Used)

*   **React.js** - ইউজার ইন্টারফেস তৈরির জন্য।
*   **React Router DOM** - পেজ নেভিগেশন এবং এসপিএ (SPA) নিশ্চিত করার জন্য।
*   **Tailwind CSS & DaisyUI** - দ্রুত এবং কাস্টম ডিজাইনের জন্য।
*   **Recharts** - অ্যানালিটিক্স পেজে পাই চার্ট প্রদর্শনের জন্য।
*   **React Toastify** - ইউজার ইন্টারঅ্যাকশনের পর নোটিফিকেশন দেখানোর জন্য।
*   **React Icons** - প্রফেশনাল আইকন ব্যবহারের জন্য।
